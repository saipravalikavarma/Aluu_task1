# ALU_Task1 – Arithmetic Logic Unit

## Internship Task – CODTECH

### 📌 Objective
Design a 4-bit ALU supporting:
- Addition
- Subtraction
- AND
- OR
- NOT

### 🛠 Tools Used
- Verilog HDL
- Vivado / EDA Playground

### 📂 Project Structure
- src/alu.v → ALU Design
- tb/alu_tb.v → Testbench
- simulation/ → Output waveform
- report/ → Project report PDF

### ⚙️ ALU Operations Table

| Select | Operation |
|--------|-----------|
| 000    | Addition  |
| 001    | Subtraction |
| 010    | AND |
| 011    | OR |
| 100    | NOT |
